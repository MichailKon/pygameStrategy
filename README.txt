Инструкция:

Escape с начального экрана - выход из игры
Escape в процессе игры - вернуться к начальному экрану
N в начальном экране - начать игру
W при нажатии на город - спавн воина(стоимость = 5)
A при нажатии на город - спавн лучника(стоимость = 10)
S при нажатии на город - спавн щитовика(стоимость = 7)
R при нажатии на деревню или вражеский город - захватить
Enter в процессе игры - следующий ход( = Skip)
P при нажатии на поле - вспахать(стоимость = 5)
C при нажатии на лес - вырубить(стоимость = 2)
L при нажатии на клетку - поставить лесорубню(стоимость = 15)
M при нажатии на клетку - поставить мельницу(стоимость = 20)

При нажатии на клетку отображается вся информация о клетке для текщуго игрока
