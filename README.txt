Инструкция:

Escape с начального экрана - выход из игры
Escape в процессе игры - вернуться к начальному экрану
N в начальном экране - начать игру
W при нажатии на город - спавн воина
    стоимость = 5,
    уровень города >= 1
A при нажатии на город - спавн лучника
    стоимость = 10
    уровень города >= 3
S при нажатии на город - спавн щитовика
    стоимость = 7
    уровень города >= 2
R при нажатии на деревню или вражеский город - захватить
Enter в процессе игры - следующий ход( = Skip)
P при нажатии на поле - вспахать
    стоимость = 5
    прокачка = +3
C при нажатии на лес - вырубить
    стоимость = 2
    прокачка = +2
L при нажатии на клетку - поставить лесорубню
    стоимость = 15
    +1 за каждый лес в округе
M при нажатии на клетку - поставить мельницу
    стоимость = 20
    +2 за каждое поле в округе

При нажатии на клетку - отображается вся информация о клетке для текщуго игрока
Если при этом на клетке есть юнит - отображается куда этот юнит может сходить и кого проатачить.
Чтобы сходить и проатачить - нажмите на юнита и на клетку куда нужно сделать действие
